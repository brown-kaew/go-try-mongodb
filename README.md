# go-try-mongodb

## Init module
```bash
go mod init github.com/brown-kaew/go-try-mongodb    
```

## Init server.go main package
```bash
touch server.go
```

## Load dependency after import public package
```bash
go mod tidy    
```

## Run
```bash
go run server.go
```

## Build
```bash
go build -o ./out .
```
